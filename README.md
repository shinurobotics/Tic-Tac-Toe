# Tic-Tac-Toe Game 🎮
A Python-based Tic-Tac-Toe game developed as part of my programming learning journey.

# About
This project implements a terminal-based Tic-Tac-Toe game using Python. Two players take turns placing X and O on a 3×3 game board. The game checks for winning combinations across rows, columns, and diagonals, and displays a Game Over screen when a winner is found or when the game ends in a draw.
The project was built to strengthen my understanding of programming fundamentals, game logic, problem-solving, and algorithm design.

# Features
- 3×3 Tic-Tac-Toe board
- Two-player gameplay
- Player turn switching (X and O)
- User input for row and column positions
- Dynamic board updates after every move
- Win detection system
- Row, column, and diagonal winner checking
- Draw detection system
- Game Over screen
- Winner announcement
- Terminal-based interface
